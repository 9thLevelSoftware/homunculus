<task id="05-02-2">
<title>acceptance.py + autonomy-report + autonomy-accept CLI</title>
<action>
1. `homunculus/autonomy/acceptance.py`:
   - `validate_acceptance(report: AutonomyReport, *, soak_branch: str, workspace_root: Path) -> AcceptanceVerdict`
   - Six predicates (one per `SC1..SC6`):
     - `_check_uptime(report) -> CriterionResult` — uptime ≥ `timedelta(days=7)`
     - `_check_self_directed_tasks(report) -> CriterionResult` — `self_directed_tasks_completed >= 10`
     - `_check_lora_merged(report) -> CriterionResult` — `loras_merged >= 1` AND `current_base_generation > initial_generation`
     - `_check_tests_pass() -> CriterionResult` — re-run `unittest discover` at acceptance time, exit 0
     - `_check_metrics_stable(report) -> CriterionResult` — `patch_success_rate_trend >= -0.02` AND (coverage_trend is None OR `coverage_trend >= -0.02`)
     - `_check_no_human_intervention(workspace_root, soak_branch) -> CriterionResult` — `git log --author` on soak branch; reject if any commit author doesn't match agent pattern (grep `commit_to_source` in `task_runner/runner.py` for the exact author string used)
   - Each predicate fills `CriterionResult.evidence` with human-readable proof and `raw` with the raw metric dict.
   - `overall = "PASS"` iff every `passed is True`.

2. CLI additions in `homunculus/cli.py`:
   - `autonomy-report` — calls `generate_report`, prints human table OR `--json` blob. Supports `--since DAY-N` to filter episodes by relative day.
   - `autonomy-accept` — flags: `--config`, `--soak-log PATH`, `--output PATH`, `--soak-branch NAME`. Calls `generate_report` then `validate_acceptance`. Writes a markdown report at `--output` with per-criterion table. Exit 0 only if `overall == "PASS"`.

Markdown acceptance report template: `# Phase 5 Acceptance\n\n**Overall**: PASS/FAIL\n\n## Criteria\n\n| ID | Name | Status | Evidence |\n|---|---|---|---|\n...`
</action>
<verify>

- `python -m homunculus.cli autonomy-accept --config homunculus.toml --output /tmp/acc.md --soak-branch phase-5/test` runs end-to-end; writes markdown; exit code matches `overall`
- All six criterion predicates callable individually from Python for fixture tests
</verify>
<done>Acceptance predicates cover all 6 success criteria with evidence; `autonomy-report` and `autonomy-accept` CLI commands work end-to-end.</done>
